# Youtubecrawling


---

Selenium, Chromedriver 사용해 크롤링 해보는 프로젝트
### 기술 스택

- Selenium Server 4.9
- Java 11
- ChromeDriver 112.0.5615.49

### 기능

---

- 유튜브 인기 급상승- 음악 크롤링
